## Dependencies
 - [Python 3.7](https://www.python.org/downloads/)
 
## Install dependencies

 1. Create one  virtualenv with python 3.7:
    ```
    virtualenv -p python3.7 env
    ```
 2. Install Python dependencies:
    ```
    make requirements-dev
    ```

## Run the project

```
make run
```
## Run the tests

```
make test
```
